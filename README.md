# An easy-to-use Among Us cheat menu with a simple GUI and lots of useful modules.

# Windows Installation
For Steam or Itch.io: Download TerraMenu-VERSION-Steam-Itch.zip.
For Microsoft Store, Epic Games Store, or Xbox App: Not supported by TerraMenu. For these platforms, go to the original MalumMenu here.
Open the zip file you have just downloaded and copy all its contents.

Paste these files directly into your Among Us game folder:

Steam: Right-click Among Us in your Library → Click Manage → Click Browse local files.
Itch.io: Open the Itch.io app → Right-click Among Us in your Library → Click Manage → Click Open folder in Explorer.
Epic Launcher: Not currently supported by TerraMenu.
Microsoft Store: Not currently supported by TerraMenu.
Xbox App: Not currently supported by TerraMenu.
Launch Among Us as you normally would. You should see a console window appear, installing the mod's requirements.

Wait for the console window to finish the installation.

After installation, Among Us will automatically open with TerraMenu successfully installed.

By default, you can toggle the cheat GUI on by pressing DELETE on your keyboard.
If the installation doesn't work, check out our FAQ.

# Linux Installation
Run Among Us under Proton (or Wine).

In Steam: Right-click Among Us in your Library → Click Properties → Click Compatibility → Enable Force the use of a specific Steam Play compatibility tool.

Test different Proton versions if you're having issues launching the game.

Set up BepInEx (the framework TerraMenu is built upon).

Follow the official Proton / Wine setup guide found here.

If you are using Proton with Steam, specify the DLL override:

In Steam: Right-click Among Us in your Library → Click Properties → Click General → Click Launch Options.

Add this to your launch options:

WINEDLLOVERRIDES="winhttp.dll=n,b" %command%
After that, continue with the Windows installation steps found here.

Fix crashes or errors (like Unable to execute IL2CPP chainloader).

In Steam: Right-click Among Us in your Library → Click Properties → Click General → Click Launch Options.

Set your launch options to:

PROTON_NO_ESYNC=1 PROTON_USE_WINED3D=1 WINEDLLOVERRIDES="winhttp.dll=n,b" %command%
